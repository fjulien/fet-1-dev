# FeT-1-DeV

Je suis Développeur fullstack principalement sur les technos JavaScript/TypeScript. J'ai choisi ce métier par passion après avoir travaillé en tant qu'Électrotechnicien après une dizaine d'années. Je mets l'accent sur la qualité, la performance, la facilité de maintenance et la simplicité dans mon travail. Je ne néglige jamais les besoins du client car le but ultime du code est toujours l'utilisateur final. 

Je m'assure également de rester à jour sur les dernières technologies en effectuant une veille régulière, tout en élargissant mes connaissances dans tous les domaines.

Et surtout, je m'efforce de suivre la philosophie KISS _(Keep it Simple, Stupid)_.
