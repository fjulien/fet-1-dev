# Les Variables

<Img src="/images/posts/variables.png" alt="Boites de variable"></Img>

<TitleSection title="TLDR"></TitleSection>

- Déclarer une variable avec `var`, `let` et `const`.
- `const` est nom réafectable.
- Assigner une valeur avec `=`.
- Préférable d'utiliser `const`, si non réafection sinon `let` et `var` à éviter.

<TitleSection title="Plus de détail"></TitleSection>

Les variables sont au coeur de tout programme en JavaScript, elles permettent de stocker des informations pour les utiliser plus tard dans votre code.
Il existe différentes façons de les déclarer `var`, `let` et `const`, utiliser les bons mots clés vous permettra de mieux gérer les portées et les réaffectations pour éviter les erreurs.

L'assignement se fait simplement avec l'opérateur `=`.

On n'ai pas obligé d'affecter la variable lors de ça déclaration

Apprendre à bien utiliser les variables est une étape cruciale pour devenir un développeur JavaScript expérimenté.

<TitleSection title="Fait voir du code"></TitleSection>

```javascript
/* EXAMPLE AVEC VAR */
var nomVariable1; // Déclaration de nomVariable1 sans assigantion
nomVariable1 = 'valeur'; // Assignation de nomVariable1
nomVariable1 = 42; // Réaffectation de nomVariable1

/* EXAMPLE AVEC LET */
let nomVariable2 = 'autre valeur';
nomVariable2 = 'Nouvelle valeur'; // Réaffectation de nomVariable2

/* EXAMPLE AVEC CONST */
const nomVariable3 = 'encore une autre valeur';
nomVariable3 = 'Nouvelle encore autre valeur'; // Cela génére une erreure car déclaré avec const
```
